# 功能介绍

# 基础架构

  • SQL脚本：一个后缀为.sql 的文件
  • 解析器：在SQL脚本中编写好的规则已经SQL语句，会首先被解析器解析为ezasse对象
  • 校验器：解析器将SQL解析后，将每个代码块传递到校验器，校验器会按照校验行的规则进行判断，以下SQL脚本是否会被执行
  • 执行器：不同的数据库类型，拥有不同的执行器，主要作用是将SQL脚本在目标数据节点上进行执行
